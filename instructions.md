# Instructions
These instructions are meant as prompts for maximum reproducibility.

## Setup before implementation

1. Ensure project guidance files are present:
   - `requirements.md`
   - `implementation.md`
   - `instructions.md`
2. If this repo is used as a submodule, initialize it first:
   - `git submodule update --init --recursive agents-md`
3. Validate that requirements and implementation rules do not contradict each other.

## Prompt shortcuts

- **initial implementation**: `Please implement this project`
- **implementation of additional requirements**: `Please implement new requirements`
