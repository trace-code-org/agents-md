# Codex Skill

To install the req-trace skill in your local codex-environment, you have to give codex these instructions:  
> You can find the req-trace repository here: `github.com/trace-code-org/req-trace`  
> Please install the req-trace skill from the repository. Copy only skill.md to: `~/.codex/skills/req-trace/SKILL.md`  
> Please copy the codex/template.md file to `~/.codex/AGENTS.md`. If the file allready exists, merge them.
