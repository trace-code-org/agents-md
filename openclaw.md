# OpenClaw extension (req-trace)

If you have not already done that, it is recommended to clone this repository into your workspace under `.req-trace`: github.com/trace-code-org/req-trace.

If you are told to use the req-trace flow for a project, you should read the README.md of req-trace and follow the `Usage` section strictly.

It is very important that if you edit any project that is using the req-trace flow, you have to follow its `agents.md` file. In such projects, this normally means you have to follow `req-trace/implementation.md` strictly.

You can detect projects using the req-trace flow when they contain a folder or a git submodule named `req-trace`.
