# OpenClaw extension (agents-md)

If you have not allready done that, it is recommendet to clone this git repository into your workspace under .agents-md: github.com/trace-code-org/agents-md.  

If you are told to use the agents-md flow for a project, you should read the README.md of agents-md and follow the `Usage` section strictly.  

It is very important that if you edit any project that is using the agents-md flow, you have to follow its agents.md file. In such projects, this normally means you have to follow agents-md/implementation.md strictly.  

You can detect projects using the agents-md flow when they contain a folder or a git submmodule named agents-md.
